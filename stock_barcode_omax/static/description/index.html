QR Code for Picking/Internal Transfer/Incoming Shipment/Delivery Order

Overview

This Odoo app allows to generate QR Code based on configuration parameters for Picking/Internal Transfer/Incoming Shipment/Delivery Order.

Problem Statement

  • In some urgent situation its hard to get key information from sale order.
  • We thought Scanning QR code is answer for above situation

Feature

  • Scan QR code from mobile and share with anybody you want.
  • Generate QR Code for Picking/Internal Transfer/Incoming Shipment/Delivery Order.
  • QR Code generation based on fields configuration in company.
  • Easy fields configuration for how many values user want to see in QR Code.
  • Works in multi-company environments.
